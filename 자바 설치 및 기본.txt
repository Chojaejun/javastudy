4. eclipse의 구조
   - 이클립스 우측 상단의 [open perspective]를 클릭하여 java perspective 선택
   - [ctrl + n]을 누른 후 [자바프로젝트 생성]
   - 프로젝트명은 첫 글자 대문자 띄어쓰기 하지 않고 띄어쓰는 단어의 첫 글자 대문자
   - [ctrl + n]을 누른 후 [package 생성]
   - 패키지는 소문자로.
   - 회사의 도메인 거꾸로
     예) kr.co.ezen, com.naver
   - [ctrl + n]을 누른 후 [class]생성
   - 클래스명은 첫 글자 대문자, 띄어쓰기 하지 않고 띄어쓰는 단어의 첫 글자 대문자.
   - main( ) 메서드를 생성.
     -> [main] 적은 후 ctrl + space]
   - console에 출력은 [sysout] 적은 후  
     println("hello world");
     println(11);
     println("h");
     println('h');
     println(true);
     println(false);

6. 변수
  =는 대입 연산자: 우측의 값을 좌측의 변수에 대입

3+3 = 5+2
컴퓨터 프로그래밍에서 = 은
a = b
b의 값을 a에 대입하세요.

a = 3; 3을 a에 대입하세요.
변수명(그릇의 이름) = 값 또는 변수
4 = 3; 3을 4에 대입하세요.(컴퓨터에서는 X)

a = 5;
a에 5를 대입하세요.(X) 
5를 a에 대입하세요.(O)
컴퓨터의 기준은 항상 왼쪽
c = b;

a = 3+7;
3과 7을 더하기 한 결과 값을 변수 a에 대입하세요.
