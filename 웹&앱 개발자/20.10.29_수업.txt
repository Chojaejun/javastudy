
조건문:
특정 조건을 만족했을 때, 원하는 작업수행을 보장하기 위한 제어문

단순 if (~~하면 ~~하고 그렇지 않으면 ~~한다)
if ~ else
if ~ else if ~ if(다중 if문)

switch ~ case문

조건문을 사용해야 하는 상황인지

1. in a = 4이다. a가 10이 될 때까지 sysout(a)를 실행하라.(조건문 x, 조건식 o)

2. 외부로부터 넘겨받은 정수 a가 10보다 크면, 
   sysout("ok") 실행.

3. 변수 b가 true이면, return 44;
그렇지 않으면 return 55;를 실행.

4. 변수 c가 10보다 크면 sysout('a');
그렇지 않고 c가 5보다 크면 sysout('b');
그렇지 않고 c가 1보다 크면 sysout('c');
이도저도 아니면 sysout('d');
